## 隐私数据

`vi ScrapyDemo/myconfig.py`，项目中所用到的隐私数据如下：

```md
MYSQL_HOST = 'mysql主机名'
MYSQL_DBNAME = 'mysql数据库名'

MYSQL_USER = 'mysql用户名'
MYSQL_PASSWORD = 'mysql登录密码'
MYSQL_PORT = 3306

ZHIHU_PASSWORD = '知乎密码'

# 斐斐打码平台：http://www.fateadm.com（另一家打码平台备份——http://www.chaojiying.com/）
 # 用户中心页可以查询到pd信息
ff_pd_id   = "xx"
ff_pd_key = "xx"
# 开发者分成用的账号，在开发者中心可以查询到
ff_app_id = "xx"
ff_app_key = "xx"
```
